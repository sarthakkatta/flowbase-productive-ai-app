# Flowbase Cozy UI Theme

## Mood
Flowbase should feel like a fresh, calm workspace for structured thinking: airy, modern, clean, and quietly energetic. It combines Notion-style clarity with Miro-style spatial creativity.

## Palette
- App background: `#f8f7f2`
- Surface: `#fffffb`
- Elevated surface: `#ffffff`
- Primary text: `#24201c`
- Secondary text: `#665f55`
- Muted text: `#7c756a`
- Borders: `#e7e1d6`
- Primary action: `#256f63`
- Primary action hover: `#1f5f55`
- Fresh active fill: `#e6f6e9`
- Soft hover fill: `#eef8ef`
- Warm accent: `#ff6b4a`
- Gold accent: `#ffe08a`
- Supporting icon colors: violet `#7c5cff`, sky `#00a7e1`, emerald `#00b894`, amber `#f5a524`, rose `#f04f78`, indigo `#3f6df6`, fuchsia `#bd3ff6`.

## Typography
- Use the system UI font stack.
- Keep headings clear and compact: semibold, calm, no oversized marketing type inside the product UI.
- Body text should use comfortable line height and muted warm grays.
- Letter spacing should stay at `0`.

## Spacing And Shape
- Use 8px border radius for buttons, panels, inputs, and sidebar items.
- Prefer 10px to 20px spacing steps for dense dashboard layouts.
- Avoid nested cards; use panels only for distinct tools, widgets, and repeated items.
- Keep icon buttons square and stable so collapse/expand states do not shift layout unexpectedly.

## Interaction
- Sidebar groups navigation into Workspace, Create, and System sections.
- Sidebar expands to show group labels and item labels, then collapses to icon-only navigation.
- Use Lucide icons for navigation and actions.
- Icons should be colorful but balanced against warm neutral surfaces.
- Hover states should be fresh mint fills or light neutral fills, not high-contrast flashes.
